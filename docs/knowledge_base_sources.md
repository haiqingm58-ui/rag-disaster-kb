# 知识库扩展建议：权威灾害资料清单

本文档列出建议收集的权威灾害应急资料，用于扩充本地知识库。
资料按优先级排列，便于分阶段导入。

---

## 优先级说明

| 级别 | 说明 |
|------|------|
| P0 | 核心必导，直接影响回答质量和安全性 |
| P1 | 重要补充，覆盖更多灾种和场景 |
| P2 | 锦上添花，提升全面性和专业性 |

---

## 一、地震避险（P0）

### 1.1 《地震预警信息公众使用指引（试行）》
- 来源：中国地震局（2025 年 11 月发布）
- 用途：地震预警等级（蓝黄橙红）、各场景避险指导
- 格式：官网公告，可整理为 Markdown
- 状态：已部分导入（`地震预警与应急避险指南.md`）

### 1.2 中国地震局科普资料
- 来源：https://www.cea.gov.cn/cea/kp/index.html
- 用途：地震成因、历史大地震案例、抗震建筑知识
- 建议：下载科普栏目中 PDF / 图文资料，用 MinerU 清洗后导入

### 1.3 地震应急避险手册（各地应急管理局）
- 来源：各省市应急管理局官网
- 用途：社区/家庭/学校/企业的具体避险方案
- 建议：优先收集北京、四川、云南等地震多发地区的资料

## 二、洪水与地质灾害防御（P0）

### 2.1 应急管理部防灾减灾科普
- 来源：https://www.mem.gov.cn/kp/zrzh/
- 用途：洪水、泥石流、滑坡、台风等的防御知识
- 状态：已部分导入（`洪水与地质灾害防御自救手册.md`）

### 2.2 水利部洪水防御资料
- 来源：http://www.mwr.gov.cn/
- 用途：洪水预警等级（蓝黄橙红）、堤防抢险、水库调度
- 建议：收集《防汛抗旱知识读本》等公开出版物

### 2.3 自然资源部地质灾害避险资料
- 来源：https://www.mnr.gov.cn/
- 用途：滑坡、泥石流、崩塌、地面塌陷等灾害识别和避险
- 建议：收集《地质灾害避险明白卡》系列

### 2.4 地质灾害避险明白卡
- 来源：各地自然资源部门
- 用途：简明扼要的灾害识别口诀和逃生路线图
- 特色：图文并茂、群众语言，适合基层社区使用

## 七、导入建议

### 分阶段策略

| 阶段 | 内容 | 预估文档数 |
|------|------|-----------|
| 第一阶段（已完成） | 地震 + 洪水核心避险 + 国家应急预案 | 4 篇 |
| 第二阶段（建议） | 各灾种明白卡、地方应急预案、校园指南 | 10-15 篇 |
| 第三阶段 | 国际资料、法律框架、行业预案 | 15+ 篇 |

### 导入格式建议

- PDF 文件：使用 MinerU 清洗后导入（pipeline 模式，关闭公式识别以提速）
- 网页内容：复制关键章节，整理为 Markdown 后导入
- 图片类资料（如明白卡）：手动转录为文字，或 OCR 后校核

### 质量控制

- 优先导入**官方来源**，避免民间非专业人员编写的资料
- 标注资料出处和发布时间，回答时可追溯
- 定期更新：应急预案和政策性文件每年核查一次
